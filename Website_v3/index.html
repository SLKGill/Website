Pacemaker Project

Software Development Course - McMaster University

In a group of 5, we created a full functioning simulated pacemaker that interfaces real time systems and acts like a real pacemaker. Using MATLAB Simulink I designed and implemented real-time logic that runs on a FRDM K-64 board. After many iterations of design, we reached a robust and scalable design; and successfully implemented hardware hiding and serial communication. My group and I completed the full Software Development Life Cycle with testing and documentation for the project.

  • Software Development Life Cycle
  • MATLAB Simulink

Reaction Time Tester

Embedded Systems Design Course - McMaster University

Using a STM32F429I Discovery board, I implemented software logic using interrupts to see how fast an individial can push a button after an LED light flashes. The reaction time was displayed on the LCD screen of the board, and the best score was stored in flash EEPROM. An external push button was also connected to the board to display the current and best results at any time requested.

  • C
  • Hardware Architecture

Prosthetic Hand Project

Design and Graphics Course - McMaster University

Collaboratively designed and modelled a proof-of-concept of a functional prosthetic hand using AutoDesk Inventor. After the design was finalized with CAD Simulations, it was 3D printed and assembled. The prototype involved a complex gear train (spur, worm and bevel gears) to achieve the goal of a finger pinching motion. Our hand design successfully was able to function within a set 5% error of output speed parameter.

  • AutoDesk Inventor
  • 3D Printing

Digital Logic Student Number Project

Analytical and Digital Circuits Course - McMaster University

I created a unique state transition table and K-Maps associated with the sequence of my student number (9 digits). It was determined that the design of my number could be optimized using completed 5 JK flip flops (reduced from 6, due to repeated bits). The logic was modelled using MultiSim with 5 JK flip flops, 8 logic gates, voltage supplies, a 7-segment display, a logic analyzer and a decoder chip to simulate this circuit. The display piece successfully output my student number following the logic expressions I calculated from the K-Maps.

  • Digital Logic Design
  • MultiSim
